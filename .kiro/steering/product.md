# Product Overview

This is an AI Trading LLM starter scaffold - a minimal end-to-end project for building an LLM-powered trading assistant.

## Core Purpose
- Provides trading recommendations based on stock price data and news sentiment
- Serves as a foundation for building more sophisticated trading AI systems
- Demonstrates integration between data collection, model training, and web interface

## Key Features
- Real-time stock price analysis with candlestick charts
- News sentiment integration for trading decisions
- Fine-tunable LLM models using QLoRA/PEFT adapters
- RESTful API for trading predictions
- Web-based user interface

## Important Disclaimers
- NOT financial advice - for educational/research purposes only
- Requires external API keys (OpenAI) or local model weights
- Always test with paper trading before real money
- No proprietary model weights included in repository

## Target Users
- Developers learning AI/ML in finance
- Researchers experimenting with trading algorithms
- Teams building proof-of-concept trading systems
